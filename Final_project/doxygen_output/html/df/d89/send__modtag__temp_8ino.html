Detailed Description

This file contains the code for Temperture sensor and control module.

The code connects to WiFi, subscribes to MQTT topics, reads sensor data (temperature prefrence and humidity prefrence), controls a servo motor, indicates wether to heat or cool the room and indicates temperature status using an RGB LED.

Author
Abbas Heidari, Daniel Heiðar Qasemiani & Jakob Høyer Madsen
